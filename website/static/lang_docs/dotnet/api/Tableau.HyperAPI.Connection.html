Class Connection

Connection to a Hyper server.

Inheritance
System.Object
Connection
Implements
System.IDisposable
Namespace: Tableau.HyperAPI
Assembly: Tableau.HyperAPI.NET.dll
Syntax
public sealed class Connection : Object, IDisposable
Remarks

Connection must be closed after you are done with it, either by calling Close() or implicitly with a using statement.

This class is not thread-safe, no two methods may be called simultaneously from different threads. The only exception is Connection.Cancel(), which may be safely called from a different thread.
