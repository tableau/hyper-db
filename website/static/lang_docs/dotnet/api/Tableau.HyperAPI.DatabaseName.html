Class DatabaseName

Quoted and escaped database name

Syntax
public sealed class DatabaseName : Object, IEquatable<DatabaseName>, IComparable, IComparable<DatabaseName>
Remarks

DatabaseName is implicitly convertible from string or Name, so one can pass raw unescaped names as strings or Name object to methods which expect a DatabaseName.
