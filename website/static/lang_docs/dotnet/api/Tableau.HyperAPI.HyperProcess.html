Shutdown(Int32)

Shuts down the Hyper process, throws a HyperException if there was an error stopping the process, if the process was forcefully killed after the timeout, or if the process already exited with a non-zero exit code.

Declaration
public void Shutdown(int timeoutMs = -1)
Parameters
Type Name Description
System.Int32 timeoutMs

The timeout in milliseconds.
Remarks

If timeoutMs > 0ms, wait for Hyper to shut down gracefully. If the process is still running after a timeout of timeoutMs milliseconds timeout, forcefully terminate the process and throw an exception.

If timeoutMs < 0ms, wait indefinitely for Hyper to shut down.

If timeoutMs == 0ms, immediately terminate Hyper forcefully. Does not throw if the process already exited with a non-zero exit code.
