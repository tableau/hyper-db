Class Result

Result of a query.

Inheritance
System.Object
Result
Implements
System.IDisposable
Namespace: Tableau.HyperAPI
Assembly: Tableau.HyperAPI.NET.dll
Syntax
public sealed class Result : Object, IDisposable
Remarks

Result object allows reading data row by row. NextRow() advances to the next row, IsNull(Int32) checks whether the value at the given index is NULL, and Get*(int) methods retrieve values.

Alternatively one can use GetValues(Object[]) and GetValues() to retrieve all row values as an array of objects, or GetValue(Int32) to retrieve individual values.

Column index.

Type Description
System.DateTime

Remarks

DateTime does not contain time zone information. However, the Kind property indicates whether the value represents local time, Coordinated Universal Time (UTC), or neither. For a TIMESTAMP_TZ column, the value is returned in UTC with Kind set to Utc. For a TIMESTAMP column, the value is returned unchanged with Kind set to Unspecified.

This method throws an InvalidCastException if the column is not of type Timestamp or TimestampTZ. It also throws an InvalidCastException if the value is NULL. Use IsNull(Int32) to check whether the value is NULL if the column may contain NULL values.

Remarks

DateTimeOffset is time-zone-aware. It represents a date and time value together with a time zone offset. Thus, the value always unambiguously identifies a single point in time. For a TIMESTAMP_TZ column, the value is returned in UTC with the offset set to 0.

This method throws an InvalidCastException if the column is not of type TimestampTZ. It also throws an InvalidCastException if the value is NULL. Use IsNull(Int32) to check whether the value is NULL if the column may contain NULL values.
