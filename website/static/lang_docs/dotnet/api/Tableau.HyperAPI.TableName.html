Class TableName

Quoted and escaped table name.

Inheritance
System.Object
TableName
Implements

Namespace: Tableau.HyperAPI
Assembly: Tableau.HyperAPI.NET.dll
Syntax
public sealed class TableName : Object, IEquatable<TableName>, IComparable, IComparable<TableName>
Remarks

TableName is implicitly convertible from string and Name, so one can pass raw unescaped names as strings or Name objects to methods which expect a TableName.
