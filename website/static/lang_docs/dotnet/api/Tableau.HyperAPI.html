Namespace Tableau.HyperAPI

The Hyper API classes reside in the Tableau.HyperAPI namespace.

Use HyperProcess to start local Hyper instances. Use Connection to run SQL queries, Catalog to query and manage tables and databases, Inserter to insert data.

Classes

Catalog

Class which is responsible for querying and manipulating metadata.

Connection

Connection to a Hyper server.

CreateMode

Constants which specify what should happen when connecting to a database depending on whether it exists.

DatabaseName

Quoted and escaped database name

Date

Hyper DATE value - a calendar date.

Endpoint

Network endpoint at which a Hyper server is accessible.

HyperException

Exception thrown by Hyper API methods. Note that error messages may change in the future versions of the library.

HyperException.ContextId

A context id.

Used to identify the source of an exception. Each throw statement has a unique context id that is stored in the thrown error.

HyperProcess

Local Hyper server instance managed by the library.

HyperServiceVersion

A Hyper Service version number of the form 'major.minor'.

Inserter
